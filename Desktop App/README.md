# German B1 Exam Timer

This is a Python-based timer application to help you track the time for each section of the German B1 exam.

## Features:
- Timers for Lesen, Horen, Schreiben, and Sprechen.
- Custom timer option.
- Sound notifications when the timer ends.

## How to Use:
1. Download the `.exe` file from the releases section.
2. Run the executable.

## Installation:
1. Clone or download this repository.
2. Run the `app.exe` file.
